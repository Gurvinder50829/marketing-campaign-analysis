# 📊 Marketing Campaign Analysis Project

This project analyzes marketing campaign data of companies to understand performance, sales trends, customer engagement, and profit insights using Python and data visualization tools.

---

## 🚀 Features
- Data cleaning and preprocessing
- Sales and profit analysis
- Monthly/Yearly trend visualization
- Top performing products/campaigns
- Interactive dashboards (if Streamlit/Plotly used)

---

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib / Plotly
- Streamlit (if used)
- Jupyter Notebook

---

## 📁 Dataset
Dataset contains marketing campaign records including:
- Sales
- Profit
- Product Name
- Date
- Customer Segment
